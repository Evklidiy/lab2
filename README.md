# Система управления складом автозапчастей (AutoPartsApp)

![Build Status](https://github.com/Evklidiy/AutoPartsApp/actions/workflows/build.yml/badge.svg)

Консольное приложение .NET Core для автоматизации учета на складе автозапчастей.
Проект выполнен в рамках лабораторной работы №2 (Вариант 41).

## Описание предметной области
Приложение управляет данными о:
- **Запчастях** (AutoParts): артикул, название, цена, совместимость.
- **Поставщиках** (Suppliers): контакты, рейтинг.
- **Поставках** (Supplies) и складских остатках (Stocks).
- **Продажах** (Sales) клиентам.

## Функционал
Реализованы следующие возможности с использованием Entity Framework Core и LINQ:
1.  **Выборка данных**: поиск поставщиков, фильтрация запчастей по цене.
2.  **Группировка**: анализ остатков на складе.
3.  **Объединение (Join)**: получение информации о поставках с именами поставщиков.
4.  **Изменение данных**:
    - Добавление новых запчастей и поставок.
    - Обновление цен и рейтингов.
    - Каскадное удаление записей.

## Технологии
- .NET 8.0
- Entity Framework Core
- MS SQL Server
- GitHub Actions (CI/CD)

## Запуск
1. Настроить строку подключения в `appsettings.json`.
2. Запустить проект через Visual Studio или команду `dotnet run`.
